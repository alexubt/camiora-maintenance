<task type="auto">
  <name>Task 1: Fix manifest.json and create install.js module</name>
  <files>manifest.json, app/install.js, style.css</files>
  <action>
1. Update `manifest.json`:
   - Change `"start_url": "/"` to `"start_url": "/camiora-maintenance/"`
   - Add `"scope": "/camiora-maintenance/"`
   - Keep all other fields unchanged (display, icons, theme_color, etc.)

2. Create `app/install.js` as a new ES module with these exports:

   `isIOS()` — returns true if `/iphone|ipad|ipod/i.test(navigator.userAgent)` and NOT standalone.

   `isInstalled()` — returns true if `window.matchMedia('(display-mode: standalone)').matches` OR `navigator.standalone === true`.

   Module-level variable: `let deferredInstallPrompt = null;`

   On module load, register `window.addEventListener('beforeinstallprompt', (e) => { e.preventDefault(); deferredInstallPrompt = e; showAndroidBanner(); });`

   `initInstallPrompt()` — exported function called from main.js:
   - If `isInstalled()` → return (don't nag)
   - If `sessionStorage.getItem('install_dismissed')` → return
   - If `isIOS()` → call `showIOSBanner()`
   - Android banner is shown via the beforeinstallprompt event handler (above)

   `showIOSBanner()` — creates a fixed-bottom banner element with:
   - Text: "Install Camiora: tap Share (box-arrow icon) then 'Add to Home Screen'"
   - A dismiss X button that sets `sessionStorage.setItem('install_dismissed', '1')` and removes the element
   - Uses CSS class `install-banner` for styling

   `showAndroidBanner()` — creates a fixed-bottom banner with:
   - Text: "Install Camiora for quick access"
   - An "Install" button that calls `deferredInstallPrompt.prompt()` and awaits the choice
   - A dismiss X button (same sessionStorage pattern)
   - Uses CSS class `install-banner` for styling

   Both banners are appended to `document.body` (not inside #app, so they survive route changes).

3. Add CSS to `style.css` for `.install-banner`:
   - Fixed position at bottom, full width, padding 12px 16px
   - Background: white, box-shadow for elevation, border-radius top corners 12px
   - Flex layout: text (flex:1) + action button + dismiss X
   - z-index: 1000 (above toast)
   - `.install-banner .install-action` — green button matching app theme (var(--green-dark))
   - `.install-banner .install-dismiss` — no-background X button
   - Max-width 480px centered on wider screens
  </action>
